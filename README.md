# SmarterOS Portal

Portal maestro del sistema operativo para negocios.

## Deploy en Vercel

```bash
vercel --prod
```

## Environment Variables

```
NEXT_PUBLIC_CLERK_PUBLISHABLE_KEY=pk_live_xxx
CLERK_SECRET_KEY=sk_live_xxx
NEXT_PUBLIC_SUPABASE_URL=https://xxx.supabase.co
NEXT_PUBLIC_API_URL=https://api.smarterbot.cl
```

## Dominios

- Production: app.smarterbot.cl
- Preview: smarteros-portal.vercel.app

## Stack

- Next.js 14
- Clerk Auth
- Tailwind CSS
- TypeScript
